# Key Replacer Sequencer

Python + tkinter + keyboard で動作する、キーボード入力置換シーケンサーアプリです。

単キーをトリガーとして、ホットキー送信、テキスト入力、マウスクリックなどの出力シーケンスを実行できます。キーマップを使った入力置換や、構成セットの保存・読込にも対応しています。

## 動作環境

- Python 3
- tkinter
- keyboard
- pyautogui
- pynput

依存ライブラリを入れる場合は、プロジェクト直下で以下を実行します。

```powershell
pip install keyboard pyautogui pynput
```

`keyboard` によるグローバルフックを使用するため、環境によっては管理者権限での実行が必要です。

## 起動方法

プロジェクト直下で以下を実行します。

```powershell
python main.py
```

起動すると `Key Replacer Sequencer (Multi Trigger)` ウィンドウが開きます。

## 基本的な使い方

1. `トリガー一覧` の `追加` から、実行に使う単キーを登録します。
2. 登録したトリガーを選択します。
3. `出力シーケンス` の `追加` から、実行したいアクションを追加します。
4. 必要に応じて `トリガーキーを抑止（suppress）` を切り替えます。
5. `開始（フックON）` を押すと、グローバルフックが有効になります。
6. 登録したトリガーキーを押すと、出力シーケンスが実行されます。

フック中に編集ダイアログを開いた場合、アプリは一時的にフックを停止して、編集操作が誤ってシーケンス実行にならないようにします。

## アクションの種類

`出力シーケンス` に追加できる主な種類は以下です。

| 種類 | 内容 | 例 |
| --- | --- | --- |
| `hotkey` | ホットキーを送信します | `ctrl+c`, `alt+tab` |
| `text` | 文字列を入力します | `sample_text` |
| `mouse_click` | 指定座標をクリックします | 左クリック、右クリック、複数回クリック |

`hotkey` は `キー入力で記録` から実際のキー操作で入力できます。OSショートカットはプリセットボタンや `プリセット編集...` から管理できます。

## フック操作

- `開始（フックON）`: キーボードフックを開始します。
- `停止（フックOFF）`: キーボードフックを停止します。
- `フック停止トリガー`: フック中に押すとフックを停止する単キーです。
- `有効/無効トグルキー`: 通常トリガーの有効/無効を切り替える単キーです。
- `通常トリガー無効化`: フックは維持したまま、通常トリガーだけを無効にします。

停止キー、トグルキー、トリガーキー、キーマップ関連キーは重複しないように設定してください。

## 連続実行

選択中トリガーの `出力シーケンス` で `連続実行` を有効にすると、登録したアクションを順に最後まで実行します。

`間隔(ms)` では、連続実行時のアクション間隔をミリ秒で指定できます。

## キーマップ

`キーマップ管理` では、入力キーを別のキーへ割り当てるキーマップを管理できます。

- `追加`: 新しいキーマップを作成します。
- `キーマップ変更`: キーマップ名や切替キーを編集します。
- `選択`: 使用するキーマップを切り替えます。
- `保存` / `別名で保存` / `読込`: キーマップ単体のJSONを保存・読込します。

`キーボードUI` を開くと、キーボードレイアウト上で設定を確認・編集できます。

## 保存と読込

通常の保存・読込は、分離JSON形式の `keymap_set.json` を入口にします。

- `保存`: 現在の構成セットを保存します。
- `別名で保存...`: 構成セットを別名で保存します。
- `読込...` / `読込（構成セット）...`: `keymap_set.json` を選択して読み込みます。
- `起動時に読む構成セットを指定...`: 次回起動時に読み込む構成セットを指定します。
- `Import...`: 旧形式を含む単一JSONを読み込みます。
- `Export...`: 単一JSONとして書き出します。

デフォルトでは、設定ファイルは `config/user/` 配下に作成されます。

## 表示

- `省略表示`: フック操作とトリガー一覧を中心にした小さい表示へ切り替えます。
- `フルに戻す`: 通常表示に戻します。
- `常に手前`: ウィンドウを最前面に固定します。
- `キーボードUI`: キーボードレイアウト画面を開きます。

## 注意事項

- トリガーは単キーのみです。
- フックは `suppress=True` を使用するため、設定によっては元のキー入力がアプリ外へ届かなくなります。
- フック停止トリガーを設定しておくと、フック中にすばやく停止できます。
- JSONを手で編集する場合、既存キーの削除や意味変更は避けてください。
- 実行中のシーケンスが意図しない入力を送らないよう、登録内容を確認してからフックを開始してください。
